The World Wide Web of the future.


Web 2.0

Web 2.0 sounds similar to an update to a browser or application. It technically behave in some similar manner when applied to the Web. Web 2.0 is a description of a range of web services that are known as the "second version" of the World Wide Web. It is a term that was first coined in 2004 at an O'Reilly Media conference to describe this version of the Web. Their notable characteristics include:

  • Web-based
  • User control of data
  • User-friendly and interactive environment
  • Collaborative/participatory facilities

It is easy to notice when looking at certain websites. Google Mail (now known as Gmail), Wikipedia, Flickr, Yahoo!, and many others have Web 2.0 characteristics that make them more modern compared to their older versions years ago. Web 2.0 is like the interactive version of the Web compared to the original Web, which was known as being "static", so to speak. Here is a list of examples about some of the vast amount of things that are considered to be Web 2.0:

  • Free web-based email
  • Project management tools
  • Word processing
  • Spreadsheeting
  • Digital photo processing
  • News feeds
  • Video hosting
  • Photo gallery services
  • Music and file sharing
  • Computer virus scanning

Bonus:

I have mentioned very briefly about the original Web, also known as Web 1.0. Web 1.0 is like the "static" version of the Web. It started back in 1989, which I would like the call the "early years of the Web". Web 1.0 is basically the World Wide Web, but sites were static at the time. They contained information just like any website should, but they never change. Once a website was made in Web 1.0, there wasn't much left to do. They just remained the same. Web 1.0 sites were not interactive, meaning there was no impact or contribution from visitors, the sites could not be altered, and updates were never made. Finally, Web 1.0 applications that users could not see how they work or change them.


Potential Web 3.0?

There is little to no information about Web 3.0, but once it does emerge in the distant future, it is certain that it would help the Web evolve into a much better version of the Internet that we currently know.
