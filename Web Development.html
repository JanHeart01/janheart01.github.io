Web Development

Web development is a broad term that refers to many things associated with developing web sites for the Internet (World Wide Web), or an intranet (private network). It ranges from simple plain text to web-based internet applications, and social network services. These jobs include web engineering, web content development, client-side/server-side scripting, e-commerce development, and several others. Web design is an additional job, but it is technically not the same as web development (this will be discussed shortly). Web development usually refers to writing markup and coding, the main aspects used in order to build websites.

As mentioned earlier, web development is not necessarily web design. They are not synonymous with each other because building a website and designing it are two different things, and it is true. Web developers focus on the functionality of the website, meaning they concentrate by making sure a website performs well. Web designers only focus on the overall appearance and feel of a website.

Aside from that misconception, there are a few more that I would like to address regarding web development. The next one refers to the fact that a website is never "finished". There may always be a glitch that requires attention or an element that may be added. Technology is always evolving over the years, and a website should do the same even if it may be considered to be "perfect". Finally, web developers do not only write code. They also interact with designers, illustrators, and others in order to develop a site, and are always researching new techniques and technologies in order to keep their sites up-to-date and functioning properly.

Web development can be divided to break down what developers can do: client-side scripting, server-side scripting, and database technology. Client-side scripting is creating websites, and using languages to build their layouts. These include:


  • HTML
  • CSS
  • JavaScript
  • Flash
  • jQuery
  • Microsoft Silverlight

Server-side scripting is the back-end of a website (learn more on Front-End and Back-End Development). Server-side scripting constructs frameworks in order to have the servers' databases communicate with a user's web browser. This too involves the use of languages:


  • ASP.NET
  • C
  • Java
  • PHP
  • Python
  • Ruby

Finally, database technologies basically means that a database is needed for a website to store data, so it can work properly. This set of skills does not include the use of languages, but it does include the use of relational database management systems (RDBMS), such as:


  • Oracle
  • Microsoft SQL Server
  • MySQL
  • IBM DB2
  • Apache
  • MongoDB
