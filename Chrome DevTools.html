Chrome DevTools

Chrome is more than just a browser.


Chrome DevTools

The Chrome Developer Tools (DevTools) are web authoring and debugging tools that are built into the browser Google Chrome, and provide web developers access into its internals and web application. It is a set of tools that may not be known by every user who uses Google Chrome for online browsing, so it fascinating to know what is really contained in the browser. It can be accessed by either selecting Tools > Developer Tools in the Chrome menu located on the top-right corner, or right-click on a page element and select Inspect Element.

There are shortcuts that can be used to access the DevTools on either Windows or Mac.

  • Ctrl + Shift + I opens DevTools
  • Ctrl + Shift + J does the same but also brings focus to the console
  • Ctrl + Shift + C opens DevTools in Inspect Element mode (can also toggle the Inspect Element mode if it is already open)

The shortcuts for Mac are identical to Windows' shortcuts for DevTools, but for the first two, Cmd + Opt + I and Cmd + Opt + J are used to open DevTools and open it with focus on the console, respectively. For the third one, it is Cmd + Shift + C.

As you can see, the DevTools contains eight task-oriented main groups located on top of the window, and each one focuses on a specific type of page/app information:

  • Elements
  • Network
  • Sources
  • Timeline
  • Profiles
  • Resources
  • Audits
  • Console

Chrome DevTools Tips and Tricks

Now it is not necessary to be a web developer in order to use DevTools. In fact, some of the features are easy to do without being too complicated. I will cover them individually:

  • You can rearrange the text and images on a web page, such as changing the order of elements by dragging them with the mouse and placing them in different locations.
  • You can experiment with colors via typing the colors you want to use. It is not necessary to understand the hexadecimal format in order to write colors.
  • When you type in passwords, they are hidden, but with Chrome DevTools, you can reveal the passwords by changing the input field from "password" to "text".
  • Web pages seem like they cannot be edited, but DevTools will let you edit pages inline by going to Console and typing document.body.contenteditable=true.
  • Chrome can be used as a calculator, so with DevTools, the values of previous calculations that are stored can be opened and used for lengthy calculations.
  • DevTools can extract information from a web page.
  • You can easily fake your location by clicking on Settings and place a different set of latitude/longitude values.
