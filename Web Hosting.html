Web Hosting

Web Hosting is the service providing space on the Internet for websites. Websites need to be published (or uploaded) with a web hosting service, which store website files in web servers connected to fast networks. Websites also have to have a domain name in order for it to be found online. Think of it as how you would use a street address to find a certain location. Well, the domain name can do just that on the Internet, and the address should be memorable and relevant to attract visitors.

Now web hosting has four types that happen to meet different needs for websites and customers:

Shared Web Hosting

The most common type of web hosting, shared web hosting is also the easiest for beginners. It is a service where one server is shared between customers. The advantages and disadvantages are:

  • It is cheaper, so you do not have to pay for the entire server upfront
  • It is easier to use as the server is preconfigured with the most popular options and the hosting company takes care of maintenance and security updates
  • Sharing server resources, however, can slow down all sites on a server if one site is busy
  • It is less flexible such as not being able to block a specific port or modify a core component

Website Builder

The website builder is the easiest way to build websites as it provides hundreds of templates that are customizable to create a website exactly as one envisions it. It includes a free domain name and a web hosting account. The advantages and disadvantages include:
  • It is the easiest way to get online without any additional learning required
  • It has a low monthly cost
  • Ecommerce websites can get tricky if there are too many products
  • Custom website functionality is difficult

Cloud Hosting

A Cloud hosting lands between shared hosting and dedicated hosting. Cloud servers share physical server hardware with other cloud servers, but virtualization technology splits it up for cloud servers to have their own resources and to be configured like a full server. There are no disadvantages (with the possible exception of the fourth bullet), the advantages are:
  • It is the most flexible, adapting to physical hardware problems and place sites in a reliable location
  • A website has dedicated resources
  • Fewer websites occupy the same server, leaving more resources than shared web hosting
  • Virtualization software provides freedom to make the server the way someone wants it, but may require a higher level of technical experience

Dedicated Hosting

Dedicated hosting is the type of hosting where it includes your own server and its resources for you to use only. It offers these advantages and disadvantages:
  • It is flexible when customizing the hardware and software specifications
  • All the resources guarantee performance
  • Allows full control to the user to have access to the servers and configure the setup when needed
  • Prior technical knowledge is required, placing a lot of responsibility to manage and configure the server
  • It is more expensive since the cost is not shared with other customers
